# LearnForge

AI-assisted adaptive learning platform for Grade 11/12 learners.

## Features (Phase 1)

- Curriculum-aligned topic help
- Adaptive practice questions
- Weakness detection and targeted revision
- Simple progress tracking

## Tech Stack

- **Framework:** Next.js 14+ (App Router)
- **Language:** TypeScript
- **Database:** PostgreSQL + Prisma ORM
- **Styling:** Tailwind CSS
- **Auth:** Email/password (NextAuth.js)

## Getting Started

```bash
npm install
npx prisma generate
npx prisma db push
npm run dev
```
