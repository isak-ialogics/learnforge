# LearnForge

A practice and learning platform. Study topics, answer questions, and track your progress.

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) to see the app.

## Tech Stack

- Next.js (App Router) + TypeScript
- Tailwind CSS
- Prisma + PostgreSQL
- OpenAI / Claude for AI-generated content
